How it works

1. Upload your PDF

Drop any PDF file. Everything is processed locally in your browser using Mozilla's PDF.js — your file is never uploaded to any server.

2. Text extraction

PDF.js reads each page and extracts the text content, detecting headings and paragraphs based on font size and spacing.

3. Download DOCX

The extracted text is converted to a Word-compatible DOCX file with headings and paragraphs. Open it in Word, Google Docs or LibreOffice.
