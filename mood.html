How mood palettes work

    
Each keyword maps to a set of color properties — hue range, saturation level and lightness. Combining multiple keywords blends those properties together to produce a coherent palette.

    

      

        
Pick your mood

        
Choose one or more keywords that describe the feeling you're going for. Mix temperature, lightness and style freely.

      

      

        
Generate & shuffle

        
Hit generate to get a palette. Not quite right? Shuffle creates a variation using the same mood rules but different random seeds.

      

      

        
Adjust & export

        
Click any swatch to fine-tune a color with the color picker. Export as HEX codes, CSS variables, Figma tokens or Tailwind config.
